# 00 - AWS Account Setup

This section helps a complete beginner create an AWS account safely.

## Goal

Create an AWS account, secure it, and prepare it for learning without unnecessary risk.

## Step-by-step

1. Go to the AWS sign-up page and create an account.
2. Add a payment method.
3. Verify phone and identity details.
4. Sign in as the root user one time for initial setup.
5. Enable multi-factor authentication (MFA) for the root user.
6. Open the billing dashboard and create a budget alert.
7. Create an IAM admin user for daily work.
8. Stop using the root user except for account-level tasks.

## What to configure first

- `MFA` for the root user
- `IAM` admin user
- billing alerts
- free tier awareness
- preferred AWS region

## Why this matters

Beginners often skip security and billing controls. That is a mistake. The first AWS lesson should be account safety and cost visibility.

## Recommended outcome

By the end of this section, you should have:

- one active AWS account
- MFA enabled
- one IAM admin user
- budget alerts configured
